# 键操作

---

* ## 列出所有 KEY

```
KEYS *

KEYS name*

KEYS user?
```
有3个通配符 *, ? ,[]+
*: 通配任意多个字符
?: 通配单个字符
\[]: 通配括号内的某1个字符
