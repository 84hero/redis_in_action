# 键操作

---

* ## 列出所有 KEY

```
KEYS *

KEYS name*

KEYS user?
```

有3个通配符 _, ? ,\[\]+      
_: 通配任意多个字符  
?: 通配单个字符  
\[\]: 通配括号内的某1个字符

* ## 删除 KEY

```
DEL key1

DEL key2 key3 keyN
```
